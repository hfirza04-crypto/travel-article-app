## ⚙️ Installation

Clone repository

```bash
git clone https://github.com/hfirza04-crypto/travel-article-app.git
```

Go to project

```bash
cd travel-article-app
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
