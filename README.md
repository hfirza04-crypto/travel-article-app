# ⚙ Installation

Clone repository

```bash
git clone https://github.com/hfirza04-crypto/travel-article-app.git
```

Masuk ke folder project

```bash
cd travel-article-app
```

Install dependencies

```bash
npm install
```

Menjalankan project

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production

```bash
npm run preview
```
